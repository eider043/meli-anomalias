# meli-anomalias

Proyecto para detección de anomalías en precios (estructura inicial).

Estructura:
- data/: datos crudos (no subir `data/precios_historicos.csv`)
- notebooks/: notebooks de exploración
- src/: código fuente (carga, modelos, evaluación)
- outputs/: predicciones y métricas
- main.py: script principal
